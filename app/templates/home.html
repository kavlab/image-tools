Добро пожаловать в приложение для работы с изображениями на основе Django!


Приложение представляет собой удобный инструмент для работы с изображениями, предоставляя две полезные утилиты:

  • Анализатор цветов:

    Эта утилита позволяет загрузить изображение и автоматически определить 5 основных цветов на нем. Полученные цветовые коды могут быть использованы в CSS и HTML.

    Может быть полезно:

    • Дизайнерам: для анализа и выбора цветовой схемы проекта.
    • Веб-разработчикам: для быстрого определения цветов изображений в процессе верстки сайтов.
    • Создателям мобильных приложений: для создания тем с выбранными цветами в приложениях.

  • Создание PDF файлов:

    Вторая утилита позволяет загрузить несколько изображений и создать из них многостраничный PDF-файл. Весь процесс обработки изображений происходит в оперативной памяти сервера, при этом файлы не сохраняются на диске.

    Это удобно для пользователей, желающих быстро создать один PDF файл из нескольких отсканированных изображений без необходимости хранения на сервере.
